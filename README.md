# Dependencies

```
# react dependencies
npm install react react-dom -S

# webpack and babel dependencies
npm install webpack webpack-cli webpack-dev-server -D
npm install @babel/core @babel/preset-env @babel/preset-react babel-loader html-loader html-webpack-plugin -D

# eslint dependencies
npm install eslint-plugin-react@latest eslint-config-standard@latest eslint@^7.12.1 eslint-plugin-import@^2.22.1 eslint-plugin-node@^11.1.0 eslint-plugin-promise@^4.2.1 -D
npm install prettier eslint-plugin-prettier eslint-config-prettier -D
```
